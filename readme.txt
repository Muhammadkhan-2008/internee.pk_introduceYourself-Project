===========================
INTERNEE PROFILE THEME
===========================

Theme Name: Internee Profile
Version: 1.0.0
Author: Muhammad Khan
Description: A professional custom WordPress theme built for Internee.pk Virtual Internship Program

===========================
INSTALLATION
===========================

1. Extract the theme folder to: wp-content/themes/internee-profile/
2. Go to WordPress Admin Dashboard
3. Navigate to Appearance > Themes
4. Activate "Internee Profile" theme
5. Set your site homepage to use the front page template

===========================
TEMPLATE FILES
===========================

- style.css: Main stylesheet with all theme styles
- functions.php: Theme setup and enqueue scripts/styles
- header.php: Header template with navigation
- footer.php: Footer template
- front-page.php: Homepage template (main profile page)
- index.php: Default template for posts/pages
- js/main.js: JavaScript for interactive features

===========================
CUSTOMIZATION TIPS
===========================

1. Change Site Title:
   - WordPress Admin > Settings > General > Site Title

2. Add Custom Menu:
   - WordPress Admin > Appearance > Menus
   - Create new menu and assign to Primary Menu location

3. Change Colors:
   - Edit style.css
   - Look for :root {} section with color variables

4. Add More Content:
   - Edit front-page.php to add sections
   - Use the provided card and grid classes for layout

===========================
REQUIREMENTS
===========================

- WordPress 5.0 or higher
- PHP 7.4 or higher
- No plugins required (works standalone)
